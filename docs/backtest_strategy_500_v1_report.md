# 500티커 전략 백테스트 v1

이 문서는 현재 프론트의 `전략 신호 → 종목 후보 → 단일 티커 상세 백테스트` 화면에서 바로 사용할 3개 전략을 고정한다.

## 공통 전제

- 검증 범위: 최근 1년, 백엔드 v1 로컬 parquet 기준
- 거래비용: 10bp
- 전략 형식: 500티커 또는 AI 커버리지 티커 중 상위 후보를 고르고, 단일 티커 상세에서는 해당 티커가 전략 포트폴리오에 편입된 기간만 보유한 결과를 보여준다.
- 포트폴리오 검증 지표와 단일 티커 상세 지표는 성격이 다르다.
- DB write, save-run, 신규 학습 없이 로컬 산출물만 사용한다.

## 전략 1. 지표 로테이션 500 v1

- 전략 ID: `indicator_rotation_500_v1`
- 목적: AI 없이 가격과 보조지표만으로 동작하는 500티커 기준선
- 사용 지표: 60일 추세, 20일 추세, MACD, RSI
- 편입: 점수 상위 20개
- 노출: 총 60%, 동일가중
- 리밸런싱: 3거래일마다 재선정
- 필터: RSI 45 이상, MACD -0.5% 이상

검증 수치:

| 항목 | 값 |
| --- | ---: |
| 전략 수익률 | 79.56% |
| 동일가중 시장 | 24.02% |
| 전략 MDD | -5.82% |
| 동일가중 시장 MDD | -7.70% |
| MDD 개선폭 | +1.88%p |
| 전략 Sharpe | 3.55 |
| 검증 티커 | 500 |

## 전략 2. AI 라인+밴드+지표 500 v1

- 전략 ID: `ai_hybrid_rotation_500_v1`
- 목적: AI line, AI band, 가격 보조지표를 함께 쓰는 혼합 전략
- 사용 AI: 1D Line, 1D Band
- 보조 지표: 60일 추세, 20일 추세, MACD
- 편입: 점수 상위 20개
- 노출: 총 60%, 동일가중
- 리밸런싱: 3거래일마다 재선정
- 라인 조건: safe line rank 65% 이상
- 밴드 조건: 하방 밴드와 밴드 폭을 위험 조정 점수로 반영

| 동일가중 시장 MDD | -7.83% |
| MDD 개선폭 | +1.52%p |

## 전략 3. AI 라인 단독 500 v1

- 전략 ID: `ai_line_only_rotation_500_v1`
- 목적: AI 지표 하나만으로 성능이 나오는지 확인하는 단일 AI layer 전략
- 사용 AI: 1D Line만 사용
- 미사용: AI Band, MA, RSI, MACD
- 편입: line 점수 상위 20개
- 노출: 총 45%, 동일가중
- 리밸런싱: 10거래일마다 재선정
- 라인 조건: safe line rank 65% 이상

## 프론트 연결 API

- 전략 목록: `GET /api/v1/strategies`
- 전략 신호 스캔: `GET /api/v1/strategies/{strategy_id}/scan?limit=500`
- 단일 티커 상세 백테스트: `GET /api/v1/strategies/{strategy_id}/backtest/{ticker}`

프론트는 위 3개 전략에 대해서는 백엔드 API를 사용한다. 기존 12개 subset 계산 방식은 legacy 전략에서만 유지한다.

## 판정

- 전략 1은 AI 없는 기준선으로 채택한다.
- 전략 2는 AI line+band+지표 혼합 전략으로 채택한다.
- 전략 3은 AI line 단독 전략으로 채택한다.
- AI band 단독 전략은 현재 단독 성능이 충분하지 않아 이번 3전략 구성에서 제외한다.
